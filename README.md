Ducttape Cloud Images
=====================

This repository contains OCI artifacts of Linux distributions

Different [branches exist](https://github.com/ducttape-infra/cloud-images/branches) for each of the upload images
  - [Alpine](https://github.com/ducttape-infra/cloud-images/tree/alpine)
  - [Fedora](https://github.com/ducttape-infra/cloud-images/tree/fedora)
  - [CentOS](https://github.com/ducttape-infra/cloud-images/tree/centos)
  - [Debian](https://github.com/ducttape-infra/cloud-images/tree/debian)
  - [Ubuntu](https://github.com/ducttape-infra/cloud-images/tree/ubuntu)
  - [AlmaLinux](https://github.com/ducttape-infra/cloud-images/tree/almalinux)
  - [RockyLinux](https://github.com/ducttape-infra/cloud-images/tree/rockylinux)
  - [OpenSUSE](https://github.com/ducttape-infra/cloud-images/tree/opensuse)
  - [FreeBSD](https://github.com/ducttape-infra/cloud-images/tree/freebsd)
